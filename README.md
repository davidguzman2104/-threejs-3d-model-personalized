# Three.js + Mixamo FBX (Skin + 5 Animaciones) con Botones y Teclado

Proyecto básico para cargar un personaje de **Mixamo** en formato **FBX** (con skin) y reproducir **5 animaciones** usando **Three.js**, cambiando entre ellas con **botones (1–5)** o **teclas (1–5)**.

---

## ✅ ¿Qué hace este proyecto?

- Carga **1 modelo base**: `skin.fbx` (**With Skin**).
- Carga **5 archivos FBX de animación**: `animacion1.fbx ... animacion5.fbx` (**Without Skin**).
- Reproduce animaciones con transición suave (crossfade).
- Incluye un escenario sencillo: **luces + piso + grid**.
- Permite **acercarte** al personaje con **OrbitControls (zoom)**.
